# Learning Sass

## Getting Started

First check out the project and install its dependencies:

```bash
$ git clone https://github.com/spilth/learning-sass.git
$ cd learning-sass
$ bin/setup
```

Then to generate the CSS file:

```bash
$ yarn run sass
```
